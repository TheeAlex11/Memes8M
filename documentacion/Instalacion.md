# Instalación y Ejecución del Proyecto Memes 8M

## 1. Requisitos del sistema

Para poder compilar y ejecutar el proyecto es necesario disponer de:

- **Java Development Kit (JDK) 25** o superior
- Sistema operativo **Windows, Linux o macOS**
- Terminal o consola de comandos
- Espacio libre mínimo: **50 MB**

### Librerías necesarias

El proyecto utiliza las siguientes librerías externas incluidas en el repositorio:

- `json-20251224.jar` → lectura de archivos JSON
- `junit-jupiter` → ejecución de tests

No es necesario descargar dependencias adicionales ya que todas se encuentran en la carpeta `lib`.

---

## 2. Descarga del proyecto

El proyecto puede descargarse de dos formas.

### Opción 1: Descargar ZIP

1. Acceder al repositorio del proyecto.
2. Pulsar en **Code → Download ZIP**.
3. Extraer el contenido en una carpeta del equipo.
